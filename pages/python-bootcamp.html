Python Bootcamp Series

Written on February 24th, 2019 by Eric Englin

What We Learn

This bootcamp is meant to serve as a basic tutorial that builds a base of programming knowledge. This allows us to start doing more complicated projects under real-world conditions.

Bootcamp Curriculum:

  • Get Everything Installed: We don't expect any previous knowledge. Accordingly, installing everything can be one of the trickiest parts of getting into data science projects.
  • Adding a Dataset: Once we have everything loaded, we look into how to load an actual dataset and make our first graphs
  • Loops and Functions: When working with big datasets, it can be extremely useful to make programs that will do a lot of the repetitive work for you. Loops and functions allow us to save a ton of time that would be spent writing the same line of code.
  • Basic Machine Learning: We will end with building a basic model that will predict who would live or die on the Titanic.
